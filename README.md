# ☕ Kawkab EL Saada - Frontend

A modern and responsive frontend website for **Kawkab EL Saada Café**, designed to provide an attractive and user-friendly experience across all devices.

## ✨ Features

* Responsive Design
* Modern & Clean User Interface
* Smooth Navigation
* Interactive Sections
* Optimized for Desktop, Tablet, and Mobile
* Well-Organized Code Structure
* Dark Mode

## 🛠️ Built With

* HTML5
